# 🎭 角色外观设置指南

## 创建新角色时的外观描述填写说明

### 📝 创建角色时的外观设置

**最新版本**：创建新角色时**不再需要手动填写外观描述**！

系统会自动为新角色设置"默认外观"，您可以稍后通过"装饰角色"功能进行详细的视觉定制。

### 💡 推荐填写内容

#### 简单填写（推荐新手）
```
默认外观
```

#### 描述性填写
```
活泼可爱的小女孩
帅气阳光的少年
优雅大方的少女
```

#### 详细描述
```
穿着校服的活泼学生
身穿礼服的优雅公主
戴着帽子的可爱少年
```

### 🎨 外观系统的完整流程

#### 1. 创建角色阶段（简化版）
```
角色名：小明
性别：男
```
**外观自动设置为：默认外观**

#### 2. 装饰角色阶段（推荐）
创建角色后，通过"装饰角色"功能进行详细设置：

- **肤色选择**: 默认、白色、浅黄、小麦色、古铜色、深色
- **服装系统**:
  - 上衣：T恤、衬衫、毛衣、夹克、西装上衣、卫衣、连帽衫
  - 下衣：牛仔裤、休闲裤、运动裤、短裤、裙子、短裙
  - 连衣裙：连衣裙、晚礼服、小礼服、长裙、短裙
  - 鞋子：运动鞋、皮鞋、凉鞋、靴子、高跟鞋、拖鞋
- **饰品系统**:
  - 帽子：棒球帽、贝雷帽、礼帽、鸭舌帽、渔夫帽、牛仔帽
  - 眼镜：墨镜、圆框眼镜、方框眼镜、细框眼镜、蛤蟆镜
  - 项链：项链、吊坠、珠链、锁链、十字架

### 🔄 外观数据格式

系统会自动将你的装饰选择转换为如下格式保存：

```
肤色:默认,上衣:T恤,下衣:牛仔裤,连衣裙:无,鞋子:运动鞋,帽子:棒球帽,眼镜:无,项链:无
```

### 💡 使用建议

1. **初始外观字段**: 可以简单填写"默认外观"或简单的描述文字
2. **详细定制**: 主要通过"装饰角色"功能进行可视化设置
3. **保存机制**: 装饰后的外观会自动保存，下次登录时依然保持

### 🎯 快速开始

1. **创建角色**: 只需填写角色名和性别（外观自动设置）
2. **进入游戏**: 点击"角色管理" → "装饰角色"
3. **自由定制**: 在可视化界面中选择心仪的服装和饰品
4. **保存效果**: 点击"保存装饰"应用更改

---

**记住**: 初始外观只是开始，后续可以通过丰富的装饰系统打造独一无二的角色形象！🎨✨
