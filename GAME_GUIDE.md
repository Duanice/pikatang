# 🎮 皮卡堂过家家游戏使用指南

## 📖 游戏简介

皮卡堂过家家是一款基于面向对象设计的虚拟养成游戏。玩家可以创建角色、管理账号、学习技能、积累经验，并通过虚拟货币系统进行游戏内的各种活动。

## 🚀 快速开始

### 环境要求
- **Java版本**: JDK 17 或更高版本
- **操作系统**: macOS, Linux, Windows
- **内存**: 最少 256MB 可用内存

### 安装步骤

1. **确认Java环境**
   ```bash
   java -version
   # 应该显示 Java 17 或更高版本
   ```

2. **进入游戏目录**
   ```bash
   cd /Users/duan/Documents/大作业/uml/pikachu_game
   ```

3. **编译游戏**
   ```bash
   ./compile.sh
   # 或手动编译：
   # javac -cp "lib/*:." -d bin src/*.java src/*/*.java
   ```

### 核心功能详解

#### 👤 账号管理
- **注册账号**: 创建新游戏账号
- **登录账号**: 使用已有账号进入游戏
- **修改密码**: 更改账号密码
- **修改邮箱**: 更新联系邮箱
- **注销账号**: 删除游戏账号

#### 🎭 角色管理
- **创建角色**: 为账号创建游戏角色
- **修改角色**: 更改角色信息
- **学习技能**: 掌握各种游戏技能
- **装饰角色**: 可视化定制角色外观

#### 👗 角色装饰系统
- **肤色选择**: 6种不同肤色（默认、白色、浅黄、小麦色、古铜色、深色）
- **服装系统**:
  - **上衣**: T恤、衬衫、毛衣、夹克、西装上衣、卫衣、连帽衫
  - **下衣**: 牛仔裤、休闲裤、运动裤、短裤、裙子、短裙
  - **连衣裙**: 连衣裙、晚礼服、小礼服、长裙、短裙
  - **鞋子**: 运动鞋、皮鞋、凉鞋、靴子、高跟鞋、拖鞋
- **饰品系统**:
  - **帽子**: 棒球帽、贝雷帽、礼帽、鸭舌帽、渔夫帽、牛仔帽
  - **眼镜**: 墨镜、圆框眼镜、方框眼镜、细框眼镜、蛤蟆镜
  - **项链**: 项链、吊坠、珠链、锁链、十字架
- **实时预览**: 选择服装后立即在模特身上显示效果
- **一键脱光**: 快速清除所有装饰，返回默认状态
- **提升经验**: 通过活动获得经验值
- **查看信息**: 浏览角色详细数据

#### 💰 虚拟货币系统
- **金币**: 高级货币，用于重要购买
- **银币**: 普通货币，用于日常消费
- **货币操作**: 赚取、消费、管理货币

#### 🛠️ 技能系统
- **生产技能**: 挖矿、钓鱼、种植
- **生活技能**: 烹饪、装饰
- **动作技能**: 各种动作技能
- **其他技能**: 特殊能力

## 📊 游戏数据

### 数据库文件
游戏数据存储在 `pikachu_game.db` 文件中：
- 位置: 项目根目录
- 大小: ~61KB (包含测试数据)
- 类型: SQLite 数据库

### 数据表结构
```
players          - 玩家基本信息
roles            - 角色详细信息
skills           - 技能定义
role_skills      - 角色技能关联
clothings        - 服装物品
accessories      - 配饰物品
wardrobes        - 角色衣橱
wardrobe_clothings    - 衣橱服装关联
wardrobe_accessories  - 衣橱配饰关联
```


```bash
# 查看数据库内容 (需要 sqlite3 命令)
sqlite3 pikachu_game.db
.schema players
SELECT * FROM players;
.quit
```

```bash
# 启用详细日志
```

## 🐛 常见问题

### Q: 编译失败怎么办？
**A**: 检查Java版本是否为17+，确认所有依赖文件存在
```bash
java -version
ls lib/sqlite-jdbc.jar
```

### Q: 数据库连接错误？
**A**: 删除旧数据库文件重新运行
```bash
rm pikachu_game.db
./compile.sh
java -cp "bin:lib/*" GameDemo
```

### Q: 游戏运行缓慢？
**A**: 检查系统内存，确保有足够的可用空间

### Q: 如何重置游戏数据？
**A**: 删除数据库文件并重新运行
```bash
rm pikachu_game.db
java -cp "bin:lib/*" GameDemo
```
